# Dan Ashby's Claude Skills Toolkit

A collection of Claude skills built by [Dan Ashby](https://evolvingleadership.uk) — Engineering Leader, conference speaker, and framework builder.

## What are Claude Skills?

Claude Skills are modular instruction sets that extend what Claude can do. Install one and Claude automatically uses it when relevant — no prompting required.

## Skills

| Skill | Description | Repo |
|-------|-------------|------|
| 🔍 Exploratory Testing | Structured exploratory testing sessions using heuristics and oracles. Produces chartered, documented test notes. | [danashby/Exploratory-Testing-Skill](https://github.com/danashby/Exploratory-Testing-Skill) |
| 🧭 Data-Driven Decisions | Guides you through structured questioning to surface data for informed decision making. Produces an ADR-style document with SWOT, pros/cons, and decision techniques. | [This repo](./data-driven-decisions/) |

## Installing a Skill

1. Download the skill folder
2. Go to **Settings → Capabilities → Skills** in Claude.ai
3. Upload the folder
4. Claude will use it automatically when relevant
